# 002-Recursive

## 简介
- 使用递推来计算斐波拉契数列

## 需求
- 从出生之后的第3个月起，每个月都生一对兔子,一对兔子成长到第3个月后每个月又生一对免子,假如兔子都不死，问第n个月n(<=50)的兔子总数为多少对？
- 第一个月和第二个月都是一对兔子，第三个月开始每个月都生一对兔子，可以得出公式为：
`f(n) = f(n-1) + f(n-2)` 其中:`f(1) = 1, n>=2`.

## 练习

### 题1：在墙角按照规律堆放着一些完全相同的正方体小块儿，只需要知道层数就可以计算所有小块的数量。输入：一个整数n(1 <= n <= 100),代表层数。输出：一个整数，表示这堆小块儿的总量。

- 规律：`f(n) = f(n-1) + n ` 其中：n为层数，`f(1) = 1, n>=2`.
