# 001-Recursion

## 简介

使用递归方式求解等差数列。

## 需求

使用递归方式求解等差数列1 4 7 10 13…第n项的值，要求输入n,输出第n项的值。

## 练习

### 题1：使用递归函数求解等比数列

- 描述：输入一个整数n,求出等比数列2 4 8 16 32……的第n项的值。
- 输入：一个整数n
- 输出：一个整数，等比数列的结果
- 输入样例：6
- 输出样例：64

### 题2：使用递归函数求解阶乘

(阶乘举例，5的阶乘：5！=$5*4*3*2*1$)

- 描述：输入一个整数n,求出n的阶乘结果。
- 输入：一个整数n
- 输出：一个整数，nl的结果
- 输入样例：5
- 输出样例：120
